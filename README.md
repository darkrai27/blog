# Reverse Log

Bilingual static blog (es/en) about reverse engineering, bot development, and the undocumented details of web security.

Built by an engineer. Real stories about WAFs, rate limits, browser fingerprinting, CAPTCHA bypasses, and creative workarounds that won't show up on Stack Overflow.

## Stack

- Pure HTML/CSS/JS — zero dependencies, no build step
- Bilingual via CSS visibility toggles (same HTML, both languages)
- Dark/light theme with flash-free inline init script
- oklch color space, no preprocessors, responsive breakpoints
- Ready for GitHub Pages deployment

- [Sneaker Bots](/articles/sneaker-bot.html) — 7 years of cat-and-mouse
- [Leverant (LVR) Akamai Bypass](/articles/LVR-bypass.html)

## Deploy

Push to GitHub, enable Pages from `main` root `/`. No build command needed.
